C. Vibrant Dark

Keep the dark background but inject bold, saturated color into every card. Each game mode is a vivid gradient. Dark but NOT dull. Like a neon arcade. Most dramatic departure from current grey-on-grey.

Note: All options include light + dark mode support. The choice here is about the default personality of the app — what users see first.
